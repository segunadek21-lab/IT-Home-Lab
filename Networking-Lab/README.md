# Networking Lab

## Objective

Learn common troubleshooting tools used in IT support.

## Tools Used

- Ping
- Traceroute
- Nslookup
- Wireshark

## Tasks Completed

### DNS Testing

Used nslookup to verify domain name resolution.

### Connectivity Testing

Used ping and traceroute to troubleshoot network connectivity.

## Lessons Learned

DNS issues can appear as connectivity problems even when internet access is working.
